    • Classes are fun.

    No not those kind of classes... those are boring (sometimes). I'm talking about Ruby Classes.

    No not classes that you attend. The kind you use while coding.

    Ruby classes are pretty interesting they allow us to condense our code and call instance variables (using an '@') to make our code more readable.

    An example of how to use a class would be the following:

    class Thing

    end

    Next we could define class variables using "@@". These are used inside the Class method and reign supreme over the other methods! In other words, class variables are shared between a given class and its subclasses. On the other hand class instance variables belong to only one class. Its subclasses will also be seperate (they are summoned using only one '@').

    By using these class and instance variables we are able to quickly and efficiently call methods within one another. This allows us to write code that is not only concise but also readable.
