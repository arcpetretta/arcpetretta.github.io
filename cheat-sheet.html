Hashes

Definition: A hash is like a dictionary. It uses a unique key to store values. Values are not required to be unique. A key can be any type of object.

Syntax: A hash is contained within curly brackets ({}). While there are many ways to write a hash, the most common appears as { key=>value } Key-Value pairs are separated by commas: { key1=>value1, key2=>value2 }

Best Uses: A hash is best used when the information does not need to be in a particular order, or when you don't necessarily want to use a numbered index to retrieve information. Since the key in a hash can be just about anything, you are not restricted.

Retrieving information: To get specific information out of a hash you identify the hash name you desire and call the value via the key using the following syntax: hash[key] = value. If you would like to get multiple pieces of information from the hash, you may use an iteration method.

Common Methods: .each (executes a codeblock for each key-value pair in the hash), .has_key? (returns true is the hash contains the specified key), .length (returns the number of key-value pairs in the hash), .select (returns a new hash with the key-value pairs for which the block of code is true).

Coding Examples: singles = { "1" => "one", "2" => "two", "3" => "three", "4" => "four", "5" => "five", "6" => "six", "7" => "seven", "8" => "eight", "9" => "nine"} def initialize(name, quantity) @food = {name=>quantity} end

Arrays

Definition: Arrays are objects that can store multiple values.

Syntax:Array = [value_1, value_2, value_3, value_4, value_5]

Best Uses: The best time to use a hash is when you want to store your information in a particular order.

Retrieving information: With an array it's possible to retrieve info using the index number from the array. Ex: array[0] = value_1 , array[1] = value_2

Common Methods: .each (executes a block of code for each item in the array), .map (executes a block of code for each item in the array and places it into a new array), .length (returns the number of items in the array), .include? (returns true if the specified item is in the array), .push() (pushes the item in the parentheses into the array)

Coding Examples: [1, 2, 3].length = 3["thing1", "thing2"].push("thing3") = ["thing1", "thing2", "thing3"]

Strings

Definition: A string is an object that holds "data in the form of characters." Essentially it is anything housed inside of single or double quotes (It can be any number, letter or symbol).

Syntax: "hello world", 'ball', "32", "h@t"

Common Methods: .to_s converts objects into strings. .capitalize capitalizes the first letter in a string. .upcase and .downcase are used to uppercase and downcase all letters in a string, respectively. .gsub! is used to swap out any matching string and replace it with another. .reverse is also often used to reverse the contents of a string.

Coding Examples:"hahahaha".gsub(/[a]/, '@') = h@hAh@h@"i don't know what we're yelling about!".upcase = "I DON'T KNOW WHAT WE'RE YELLING ABOUT!""Why is this backwards?".reverse

Integers

Definition: An integer is a fixed number with no decimals.

Syntax: write a number

Common Methods: .to_s (turns the integer into a string), .split()(splits a multiple-digit number into an array of strings). You can also perform mathematical operations with arithmetic operators.

Coding Example: 45.split() = ["4", 5"]
